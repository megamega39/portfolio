# portfolio
グラボ買い時チェッカー
## サービス概要
今グラフィックボードが買い時かどうかを判定するサイト

## このサービスへの思い・作りたい理由
グラフィックボードが欲しくなって今が買い時かどうかがわからず細かく調べる必要があったので
すぐに買い時かわかるサイトが有れば便利だなと思った。

## ユーザー層について
グラフィックボードが買い時かをチェックしたい人たち

## サービスの利用イメージ
ひと目見てグラフィックボードが買い時かどうかがわかる

## ユーザーの獲得について
SNS・コミュニティ活用

## サービスの差別化ポイント・推しポイント
価格コムなどで細かい情報は調べるとわかるのですが買い時かどうかはひと目見てわからないのでひと目見て買い時かがわかるのが推しポイント

## 機能候補
MVPリリース：
グラフィックボード全体の現在価格と過去の価格から買い時かどうかを判定する
人気の型番ごとに判定する
本リリース：
価格推移のチャート表示
